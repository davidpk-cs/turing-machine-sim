Turing Machine Simulator

Head Memory: _

Status: Not Running

load STRING ->load the string into the tape, must be the first line if you choose to put it in

move right|left -> move right or left 1 space

move right|left X -> move right or left x number of spaces

move right|left to end|X ->move right or left until you encounter an empty cell/ any value X

blackout -> blots out the cell, makes it unreadable

mark|unmark -> marks or unmarks the cell, it is still readable

if match|mismatch command -> run command if the heads memory matches the cell

if match|mismatch STRING|blank command -> run command if cell content matches string / is blank

else COMMAND -> can be appended to the above 2 commands if you want a command to run when the if fails
